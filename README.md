# NOME DO PROJETO

## Resumo
O aplicativo é uma solução mobile para agendamento e organização de consultas e exames, permitindo ao usuário escolher especialidade, clínica, médico, data e horário, além de anexar encaminhamentos para exames. Ele também possibilita o cadastro e gerenciamento de dependentes, centralizando os agendamentos por pessoa e facilitando a visualização e cancelamento. O sistema envia notificações automáticas de lembrete antes dos compromissos e mantém a disponibilidade de horários atualizada em tempo real.
## Atores/Personas
Paciente que tem necessidade em marcar consultas e exames, além de estar ciente desses mesmos compromissos, facilitando a organização e comunicação Paciente-Médico.
